- git --help

- git config user.name

- git config user.email

- git config --global user.name = "rochafialvin"

- git init (Inisialisasi repository git)

- git status (Melihat status file)

- git add (Mengcopy versi file yang ada di working ke stage)

- git commit -m "Pesan" (Membuat commit/checkpoint)

- git log (Melihat daftar checkpoint dengan informasi lengkap)

- git log --oneline (Melihat daftar checkpoint dengan informasi singkat)

- git revert (commit id) (Membuat commit baru sebagai penanda bahwa ada commit yang di coret)

- git reset (commit id) (Menghapus commit tanpa menghapus code)

- git reset (commit id) --hard (Menghapus commit berikut dengan code)
